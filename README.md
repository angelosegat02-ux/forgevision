# ForgeVision V9

Base definitiva del sito ForgeVision.

## File principali
- `index.html` — home principale
- `css/style.css` — stile completo responsive
- `js/app.js` — animazioni, ForgeVision Studio e Idea Generator
- `assets/` — immagini e risorse future
- `pages/` — pagine future

## Pubblicazione
Modifica i file, poi usa:

```bash
git add .
git commit -m "Aggiornamento ForgeVision V9"
git push
```

Vercel aggiornerà automaticamente il sito.
